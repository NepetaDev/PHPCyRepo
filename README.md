# PHPCyRepo

Cydia repo manager written in PHP.

The code is awful and I know it.

## Requirements:

* Linux
* Apache (or nginx, see: http://flightphp.com/install/)
* composer
* tar, ar, bzip2, gzip
* MySQL
* PHP 7

## How to install:

* create a new MySQL database
* import db.sql
* edit config.php
* run: composer install
* go to /admin
* username: admin, password: test
